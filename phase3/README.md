# Phase 3: Premium User Interface (Next.js)

## Overview
This phase implements a modern, high-end dashboard for the AI Restaurant Recommendation Service. It features a dark-themed, glassmorphic design and real-time interaction with the Phase 2 backend.

## Features
- **Modern UI**: Built with Next.js, Tailwind CSS, and Framer Motion.
- **AI Insights**: Displays natural language rationale generated by Groq.
- **Responsive Design**: Fully optimized for mobile and desktop.
- **Interactive Feedback**: Loading states, micro-animations, and error handling.

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
Ensure the Phase 2 Backend is running first!
```bash
npm run dev
```
The UI will be available at `http://localhost:3000`.

### 3. Run Tests
```bash
npm test
```

## Key Files:
- `app/page.tsx`: Main dashboard component.
- `app/globals.css`: Premium design system tokens and glassmorphism styles.
- `app/page.test.tsx`: Component-level unit tests using Jest and React Testing Library.
