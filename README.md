# AI Restaurant Recommendation Service

An intelligent restaurant discovery application that uses user preferences and LLMs to provide curated dining recommendations using the Zomato dataset from Hugging Face.

## 🚀 Project Goal
Build a high-end, AI-powered service that takes user constraints (Price, Place, Rating, Cuisine) and returns the best-fit restaurant recommendations with a clear, LLM-generated rationale.

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS / ReactJS / Next.js
- **Backend**: Python (FastAPI) / Spring Boot / Node.js (**Preferred**)
- **AI**: Groq LLM (High-speed Inference)
- **Data**: Zomato Dataset (Hugging Face)
